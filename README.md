# Web Design Agency Project

The **Web Design Agency Project** is a modern and responsive front-end web layout designed for a fictional web design agency.  
This project highlights a professional homepage layout with navigation, featured services, portfolio section, and contact area, built using core web technologies and responsive design principles.

---

## 🌐 Live Demo

Check it out here:  
👉 https://bahadirbektore.github.io/web-design-agency-project/

---

## 🚀 Project Overview

This project includes:

- A modern and professional homepage layout  
- Navigation bar with links  
- Hero banner section  
- Services and features presentation  
- Portfolio and testimonial section  
- Responsive design for mobile and desktop  
- Visually appealing typography and layout

---

## 🛠 Technologies Used

This project was built using:

- **HTML5** – Semantic markup  
- **CSS3** – Modern styling and layout  
- **Responsive design** principles
